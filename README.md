## 📡 Tecnologias

  - React
  - Vite
  - Styled Components
  - EsLint

## ⬇ Instalação
  ```bash
  # Clone este repositório
  git clone git@github.com:GustavoSrr/gitsearch.git
  # Entre na pasta gerada
  cd gitsearch
  # Instale as dependências
  npm i
  # ou
  yarn
  # Inicie o projeto
  npm start
  # ou
  yarn dev
```
